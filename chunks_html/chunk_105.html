B.2.2 Input  le structure The input  le is composed of three parts : declaration of the signature and of a prece- dence ordering over symbols, declaration of the set of axioms (symbol 'ax'), declaration of the initial set of clauses and of the symbol 'eqTheory'. The button 'Load' gives access to two templates (one for each calculus). Each tem- plate recalls the input  le structure. All you should have to do is to  ll the dotted parts, rename identi ers ending with a number (sort1, symbol2, etc) and write your own axioms and initial schematic clauses. The rest of the templates should remain unchanged. B.2.3 Get and read output The 'Output' text area contains the results of the calculation, which are either "No output" if there is no results, or a list of clauses if the calculation ended successfully.

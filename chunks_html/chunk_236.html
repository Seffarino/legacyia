ANNEXE A. DOCUMENTATION DÉDIÉE AU DÉVELOPPEUR 43 A.8 Gestion des logs L'écriture des logs est e ectuée par la fonction writeLog() de l'application Node.js ( chier /opt/audacy-webapp/app/app.js ). Les logs sont enregistrés dans le répertoire /opt/audacy-webapp/log . Ils sont orga- nisés en trois sous-répertoires : log/user_examples contient le code et les résultats des calculs d'utilisateurs ayant coché la case "I accept that this code may be used as a new example" dans le code saisi, et dont le calcul n'a pas produit d'erreurs. log/no_error contient le code et les résultats des calculs ne contenant pas la ligne indiquée précédem- ment, et n'ayant pas produit d'erreurs. En n, log/other contient le code et les résultats de tous les autres calculs. Les codes et résultats de chaque calcul sont enregistrés dans un dossier dont la convention de nommage est la suivante : adresseIP-date-heure

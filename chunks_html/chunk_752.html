CHAPITRE 4. RÉALISATIONS 21 4.3 Modules Node.js utilisés Les modules Node.js utilisés par l'application web sont listés dans le  chier de con - guration Node.js audacy/audacy-webapp/app/package.json . Les parties suivantes ex- pliquent l'utilité de chacun des modules utilisés dans le projet. 4.3.1 EJS EJS 1 est un système de templates pour les pages HTML du site. La page de déve- loppement de ce module est https://github.com/visionmedia/ejs . L'avantage d'EJS pour notre projet est d'être compatible avec Express, un autre module utilisé pour créer le serveur web, et notamment son système de vues. Ainsi, lorsque l'utilisateur demande à a cher une page, le serveur web construit la réponse dynamiquement à partir d'un template EJS contenant le code HTML de la page et des variables Javascript. La  gure 4.3.1 est un exemple simple de template EJS, a chant le nom d'un utilisateur si la variable user (un objet JSON contenant le champ name ) existe

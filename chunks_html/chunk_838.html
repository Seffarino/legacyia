EXTRAIT DE L'ARTICLE [TGRK14] 56 D.1.2.1 Possibly empty lists The signature of the theory PEL of possibly empty lists is composed of the unary function symbols car and cdr, the binary function symbol cons and the constant nil, denoting the empty list. This theory is unsorted, or, equivalently, de ned with only one sort. The theory PEL is axiomatized by the following set Ax(PEL ) of axioms : car(cons(X, Y )) = X (D.9) cdr(cons(X, Y )) = Y (D.10) cons(X, Y ) ̸= nil (D.11) cons(car(Y ), cdr(Y )) = Y ∨ Y = nil (D.12) car(nil) = nil (D.13) cdr(nil) = nil (D.14) where X, Y are universally quanti ed variables. The set G0 consists of the empty clause ⊥ and the following constrained clauses : x = y ∥ const(x, y) (D.15) x̸= y ∥ const(x, y) (D.16) car(x) = y ∥ const(x, y) (D.17) cdr(x) = y ∥ const(x, y) (D.18) cons(x, y) = z ∥ const(x, y, z) (D.19) where x, y and z are constrained variables

ANNEXE B. DOCUMENTATION UTILISATEUR 47 B.2 Input syntax The best way to learn the input syntax is to consult the examples and templates from the 'Audacy' page. See how to do this in the section "Run a computation". B.2.1 Reserved words The words 'Tops', 'Ints', 'ax' and 'eqTheory' are reserved, with the following mea- ning : The sort 'Tops' is already de ned in the prelude, in the module TOPSORT. The declaration of your sorts has to protect this module and your sorts have to be subsorts of Tops. The sort 'Ints' supports the theory of Integer O sets. See examples in [TRGK13]. The symbol 'ax' stores the set of axioms of the theory. The symbol 'eqTheory' stores the set composed of 'ax' and the set of initial schema- tic clauses. Some strings are forbidden and you won't be able to run a computation if they are present in your Maude code. These strings are "quit", "ls", "cd", "in", "pwd", "pushd", "popd", "load", and "eof"

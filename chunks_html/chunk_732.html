. Un éditeur de code Py- thon avec coloration syntaxique est intégré dans la page web. L'utilisateur exécute son programme en appuyant sur le bouton "Submit" et le résultat est a ché : si le code saisi constitue une solution à l'exercice, alors "Success !" est a ché, sinon le texte "Try again" est a ché avec une explication des erreurs sous la zone d'édition de texte. Cette interface est très similaire à ce qui est attendu dans le cadre de notre projet, mais elle ne propose pas de charger un  chier Python dans la zone d'édition de texte : il faut pour cela faire un copier-coller du texte du  chier dans l'éditeur de la page web.

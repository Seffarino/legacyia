Annexe B Documentation utilisateur Une documentation en anglais est mise à la disposition des utilisateurs sur le site. Son contenu est le suivant : 'AUDACY' means 'AUtomated Decidability And CombinabilitY'. The Audacy tool implements a complete many-sorted schematic superposition calculus for  rst-order equational theories. The tool builds the schematic form of the saturations of a super- position calculus, allowing to automatically prove decidability of theories. The tool is a prototype (proof-of-concept). It is executable online, without any warranty. The main documentation can be found in the research papers listed in the section of references. The present online documentation mainly describes the practical aspects of the user interface, input and output syntax. For all the other informations, please report to the research papers. B.1 Tool architecture (in a nutshell) The tool is executed on the server, from an input entered by a user in the 'Audacy' page

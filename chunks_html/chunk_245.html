The server  rst executes the Audacy prelude, containing all that is needed to support the syntax of input  les. Then the server executes the input  le. Finally, it executes the main Audacy code and runs the computation of the schematic saturation. More precisely, there are two versions of the main Audacy code, implementing two schematic superposition calculi. The second one is for theories extending the theory of Integer O sets [TRGK13].

DOCUMENTATION UTILISATEUR 49 If the expected duration is over and the computation is not yet  nished, you can provide your e-mail address so that the result of the computation will be sent to you by e-mail. A timer is displayed to show the elapsed time before asking for an e-mail address. If the computation time is lower than the maximal waiting time, the results are displayed on the home page. If the computation is successful, the  rst output window displays the result of the computation sent back by the server. Otherwise, the second output window displays informations about errors and warnings generated by Audacy. In this case the user is invited to revise her output and submit it again. B.3.3 How to get the results The results can be obtained thanks to a copy/paste from the display areas. If the results are sent by e-mail, the body of the e-mail contains the input and all the outputs

ANNEXE B. DOCUMENTATION UTILISATEUR 48 The 'Errors' text area indicates if there are any errors, or if a timeout occured. If there are errors, the number of warnings and the number of advisories are displayed. It also indicates if the input contains forbidden strings such as 'quit'. B.2.4 Results publication By using the web interface, the user approves that her results can be saved in the Audacy database. Those results can be published with no prior authorization from the user. However, the name of the user can appear at the time of the publication if she indicates it in the input code. B.3 Web interface B.3.1 Navigation The horizontal menu bar at the top of the pages gives access to all the pages of the application. The 'Audacy' page contains the main features of the interface : access to examples and templates, execution of a computation, output display, etc. The Documentation page is the present page.
